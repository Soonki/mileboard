Props interface:
```typescript
interface FilterDropdownProps {
  label: string;              // "ステータス", "担当者", "カテゴリ"
  options: FilterOption[];    // from extractXxxOptions
  selectedIds: Set<number | null>;  // currently selected IDs
  onToggle: (id: number | null) => void;  // called when checkbox toggled
}
```

実装:
- `useState<boolean>` で `isOpen` 管理
- `useRef<HTMLDivElement>` で `containerRef` (click outside検知)
- `useRef<HTMLDivElement>` で `listRef` (キーボードナビゲーション)
- `useState<number>` で `focusedIndex` 管理(-1 = none)
- トリガーボタン: `<button>` + label + Unicode `\u25BE` (小下向き三角)
  - `aria-haspopup="listbox"`, `aria-expanded={isOpen}`, `aria-label="{label}フィルタ"`
  - テキスト色: `selectedIds.size > 0 ? styles.active : styles.inactive`
- ドロップダウンパネル: `<div>` with `role="listbox"`, `aria-multiselectable="true"`
  - 条件付きレンダリング: `isOpen && (...)`
  - z-index: 20 (06-UI-SPEC契約)
  - max-height: 240px, overflow-y: auto
  - 各オプション: `<div role="option" aria-selected={checked}>`
    - カスタムcheckbox: 16x16px div + checked時Unicode `\u2713`
    - label text: `option.label`
    - onClick: `onToggle(option.id)` (D-04: チェック即反映、ドロップダウン開いたまま)
  - options.length === 0 の場合: `<div>選択肢なし</div>`
- click outside: `useEffect`で`document.addEventListener('mousedown', handler)` + `containerRef.current.contains`
- キーボード: トリガーボタンの`onKeyDown`で:
  - Enter/Space: toggleOpen
  - パネル内`onKeyDown`:
    - ArrowDown: focusedIndex + 1 (循環)
    - ArrowUp: focusedIndex - 1 (循環)
    - Enter/Space: onToggle(options[focusedIndex].id)
    - Escape: close
- ドロップダウン開閉アニメーション: CSS `opacity` + `transform: translateY(-4px) to translateY(0)`, `--transition-fast`

Props: なし(storeから直接取得)

実装:
```typescript
import { useMemo } from 'react';
import { useBoardStore } from '../../stores/boardStore';
import { useFilterStore } from '../../stores/filterStore';
import { extractStatusOptions, extractAssigneeOptions, extractCategoryOptions } from '../../utils/filterUtils';
import type { BacklogIssue } from '../../types/backlog';
import type { BoardData } from '../../types/board';
import { FilterDropdown } from '../FilterDropdown/FilterDropdown';
import { FilterChip } from '../FilterChip/FilterChip';
import styles from './FilterBar.module.css';
```

- `useBoardStore((s) => s.data)` で BoardData 取得
- `useFilterStore()` から statusIds, assigneeIds, categoryIds, toggleStatus, toggleAssignee, toggleCategory, removeFilter, clearAll, hasActiveFilters を取得

- `allIssues` を useMemo で計算: BoardDataの全milestones[].issuesとunassignedIssuesをflatten
  ```typescript
  const allIssues = useMemo(() => {
    if (!data) return [];
    return [
      ...data.unassignedIssues,
      ...data.milestones.flatMap((mwi) => mwi.issues),
    ];
  }, [data]);
  ```

- `statusOptions`, `assigneeOptions`, `categoryOptions` を各extract*Optionsのusememoで計算

- レンダリング:
  ```
  <div className={styles.filterBar} role="toolbar" aria-label="フィルタ">
    <div className={styles.dropdowns}>
      <FilterDropdown label="ステータス" options={statusOptions} selectedIds={statusIds} onToggle={toggleStatus} />
      <FilterDropdown label="担当者" options={assigneeOptions} selectedIds={assigneeIds} onToggle={toggleAssignee} />
      <FilterDropdown label="カテゴリ" options={categoryOptions} selectedIds={categoryIds} onToggle={toggleCategory} />
    </div>
    <div className={styles.chips}>
      {/* status chips */}
      {statusOptions.filter(o => statusIds.has(o.id as number)).map(o => (
        <FilterChip key={`status-${o.id}`} label={o.label} onRemove={() => removeFilter('status', o.id)} />
      ))}
      {/* assignee chips */}
      {assigneeOptions.filter(o => assigneeIds.has(o.id)).map(o => (
        <FilterChip key={`assignee-${o.id}`} label={o.label} onRemove={() => removeFilter('assignee', o.id)} />
      ))}
      {/* category chips */}
      {categoryOptions.filter(o => categoryIds.has(o.id as number)).map(o => (
        <FilterChip key={`category-${o.id}`} label={o.label} onRemove={() => removeFilter('category', o.id)} />
      ))}
      {hasActiveFilters() && (
        <button className={styles.clearAll} onClick={clearAll} aria-label="すべてのフィルタをクリア">
          すべてクリア
        </button>
      )}
    </div>
  </div>
  ```

- data が null の場合は null を返す(ボードデータ未読込時はFilterBarを表示しない)
