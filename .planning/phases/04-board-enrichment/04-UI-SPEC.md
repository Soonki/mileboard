### Phase 4 Dynamic Color: StatusBadge

| Property | Value | Source |
|----------|-------|--------|
| Badge background | `BacklogStatus.color` from Backlog API (hex, e.g. `#ed8077`) | D-01 |
| Badge text | Auto-contrast: `#000000` (luminance > 0.179) or `#ffffff` (luminance <= 0.179) | D-02 |
| Badge border | `transparent` when color is provided; `var(--color-border)` when color is undefined (fallback) | D-02 |
| Fallback (no color) | Existing gray badge: bg `var(--color-bg)`, text `var(--color-text-secondary)`, border `var(--color-border)` | Existing StatusBadge.module.css |

Contrast algorithm: WCAG 2.0 relative luminance (sRGB linearization with 0.179 threshold for ~4.5:1 ratio).

### Phase 4 Toggle: LaneHeader

## Interaction Contracts

### IssueCard Click --> Browser Open

| Property | Value | Source |
|----------|-------|--------|
| Trigger | Single click (onClick) | D-07 |
| Action | Open `https://{hostUrl}/view/{issueKey}` in default browser | D-06 |
| API | `openUrl()` from `@tauri-apps/plugin-opener` (direct import, not via tauriBridge) | RESEARCH.md |
| hostUrl source | `useSettingsStore((s) => s.settings.hostUrl)` | D-06 |
| hostUrl normalization | Strip `https://` or `http://` prefix before constructing URL | RESEARCH.md Pitfall 2 |
| Failure handling | Silent catch -- ripple animation confirms click visually | RESEARCH.md Open Question 2 |
| Cursor | `cursor: pointer` | D-05 |
| Hover feedback | Background transitions to var(--color-bg) over var(--transition-fast) (150ms) | D-05 |
| Click feedback | CSS `:active::after` radial-gradient ripple, 300ms ease-out, scale 0.5-->2, opacity 1-->0 | D-05, Claude's discretion |
| Accessibility | `role="link"`, `aria-label="{issueKey}をBacklogで開く"` | Derived from UX-03 |
| DnD consideration | None -- Phase 4 onClick only, Phase 5 handles DnD conflict | D-07 |

### LaneHeader Toggle

| Property | Value | Source |
|----------|-------|--------|
| Default state | Collapsed (issue count visible, breakdown hidden) | D-03 |
| Toggle trigger | Click toggle button | D-03 |
| Toggle icon | Collapsed: `\u25BC` (black down-pointing triangle), Expanded: `\u25B2` (black up-pointing triangle) | D-08, CLAUDE.md (Unicode only) |
| Toggle position | Right end of header title row | D-08 |
| Toggle visibility | Only visible when memberBreakdown has items (length > 0) | Derived from D-03 |
| Expand animation | max-height transition from 0 to 300px, opacity 0 to 1, duration var(--transition-normal) (200ms) | Claude's discretion |
| Collapse animation | max-height transition from 300px to 0, opacity 1 to 0, duration var(--transition-normal) (200ms) | Claude's discretion |
| State management | Local useState in LaneHeader (not Zustand -- only ~7 lanes, no persistence needed) | RESEARCH.md Pitfall 4 |
| Accessibility | `aria-expanded={isExpanded}`, `aria-label` toggling between "内訳を開く" and "内訳を閉じる" | Derived from BOARD-06 |
