## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Information Disclosure | boardStore.ts | accept | boardStore does not log or expose apiKey. fetchBoardData passes it via IPC only. No console.log in store. |
| T-03-02 | Spoofing | boardStore.fetchBoard | accept | Settings are read from local settingsStore (already validated in Phase 1). No external input enters boardStore directly. |
| T-03-03 | Denial of Service | boardStore.fetchBoard | mitigate | Reload button is disabled during fetch (UI layer in Plan 03). boardStore does not prevent concurrent calls, but UI prevents user trigger. |
</threat_model>
